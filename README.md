## ✨ Features
✔️ Responsive navbar with dropdowns
✔️ Product grid layout
✔️ Hover effects & animations
✔️ Mobile-friendly design
✔️ Clean semantic HTML

Simply open the index.html file in any browser - no dependencies needed!
